# Number Guessing Game 🎯

This is a simple Python-based number guessing game.

## Description
The computer generates a random number between 1 and 100.
The user has to guess the number with hints like:
- Too High
- Too Low
- Correct

## Features
- Random number generation
- Hint system
- Attempt counter

## How to Run
python main.py
